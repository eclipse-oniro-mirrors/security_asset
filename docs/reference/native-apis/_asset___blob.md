# Asset_Blob


## 概述

关键资产中使用的bytes类型，其值为字节数组。

**起始版本：** 11

**相关模块：**[AssetTypeApi](_asset_type_api.md)


## 汇总


### 成员变量

| 名称 | 描述 | 
| -------- | -------- |
| [size](#size) | uint32_t<br/>表示字节数组的大小。  | 
| [data](#data) | uint8_t \*<br/>指向字节数组的数据。  |
